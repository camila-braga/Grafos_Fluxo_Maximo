Como executar os algoritmos:

Certifique-se de que a linguagem python está instalada através do terminal, digitando:
	python --version

Uma vez instalada, acesse a pasta onde o arquivo está localizado:
	Windows: 
		cd "C:\seu\diretorio\onde\esta\o\arquivo"
	Linux/macOS: 
		cd /home/seu_usuario/seu diretório

Para executar o algoritmo de Edmonds-Karp, digite:
	python EdmondsKarp.py

Para executar o algoritmo de Dinit, digite:
	python NOME_ALGORITMO.py
